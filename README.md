# chap01

1. 컴퓨터 비전 기술이 적용된 제품

제품 예시: 스마트폰 얼굴 인식 시스템 (Face ID)

(1) 제품 기능

스마트폰의 얼굴 인식 기능은 사용자의 얼굴을 인식하여 잠금을 해제하거나 결제 인증, 앱 로그인 등을 가능하게 한다.

(2) 동작 원리

1. 스마트폰의 카메라와 센서가 사용자의 얼굴 이미지를 촬영한다.
2. 촬영된 이미지에서 눈, 코, 입 등의 특징점을 추출한다.
3. 저장된 얼굴 데이터와 현재 촬영된 얼굴 데이터를 비교한다.
4. 두 데이터가 일치하면 사용자를 인증하고 스마트폰 잠금을 해제한다.

(3) 사용된 컴퓨터 비전 기술

* 얼굴 검출(Face Detection)
* 특징점 추출(Feature Extraction)
* 패턴 인식(Pattern Recognition)
* 딥러닝 기반 얼굴 인식


2. 영상처리와 컴퓨터 비전의 차이점

영상처리(Image Processing)

영상처리는 디지털 이미지를 입력받아 이미지의 품질을 향상시키거나 변형하는 기술이다.
주로 노이즈 제거, 밝기 조절, 대비 향상, 필터링 등의 작업을 수행한다.

컴퓨터 비전(Computer Vision)

컴퓨터 비전은 이미지나 영상으로부터 의미 있는 정보를 추출하여 컴퓨터가 장면을 이해하도록 하는 기술이다.
예를 들어 사람 인식, 차량 인식, 얼굴 인식 등이 있다.

차이점 정리

| 구분    | 영상처리          | 컴퓨터 비전       |
| ----- | ------------- | ------------ |
| 목적    | 이미지 품질 개선     | 이미지의 의미 이해   |
| 처리 대상 | 픽셀 데이터        | 객체 및 장면      |
| 예     | 노이즈 제거, 밝기 조절 | 얼굴 인식, 객체 탐지 |

즉, 영상처리는 이미지 데이터를 가공하는 기술이고, 컴퓨터 비전은 이미지의 내용을 분석하고 이해하는 기술이다.


3. CMOS 센서의 원리

CMOS image sensor는 디지털 카메라와 스마트폰 카메라에서 널리 사용되는 이미지 센서이다.

동작 과정

1. 빛 수집
   렌즈를 통해 들어온 빛이 이미지 센서의 픽셀에 도달한다.

2. 광전 변환
   빛이 포토다이오드에 닿으면 광전효과에 의해 전자가 생성된다.

3. 전기 신호 변환
   생성된 전자가 전압 신호로 변환된다.

4. 디지털 신호 변환
   아날로그 신호를 디지털 값으로 변환하여 이미지 데이터를 생성한다.

CMOS 센서의 특징

* 전력 소모가 낮다.
* 읽기 속도가 빠르다.
* 스마트폰과 디지털 카메라에서 널리 사용된다.


4. 영상 해상도와 이미지 센서 개수의 관계

영상의 해상도는 이미지 센서의 픽셀 수와 직접적인 관계가 있다.

예를 들어,

* 640 × 480 해상도 → 약 30만 픽셀
* 1920 × 1080 해상도 → 약 200만 픽셀
* 4000 × 3000 해상도 → 약 1200만 픽셀

이미지 센서의 픽셀 수가 많을수록 더 많은 정보를 표현할 수 있으므로 해상도가 높아진다.
하지만 해상도가 높아질수록 저장 공간과 처리해야 할 데이터의 양도 증가한다.


5. 100×100 그레이스케일 영상 파일의 크기 계산

그레이스케일 영상은 한 픽셀이 8bit (1 byte) 로 표현된다.

따라서 영상 크기는 다음과 같이 계산할 수 있다.

영상 크기 = 가로 × 세로 × 1 byte
         = 100 × 100 × 1
         = 10,000 byte

결과: 10,000 byte


6. 100×100 컬러 영상 파일의 크기 계산

컬러 영상은 RGB 세 가지 채널을 사용한다.

* Red : 1 byte
* Green : 1 byte
* Blue : 1 byte

따라서 한 픽셀은 3 byte 이다.

영상 크기 = 가로 × 세로 × 3 byte
         = 100 × 100 × 3
         = 30,000 byte

결과: 30,000 byte
